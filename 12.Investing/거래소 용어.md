### 7. PNL (ROE %)

**가장 중요한 항목!**

#### PNL = Profit and Loss

- 손익 금액

#### ROE = Return On Equity

**자기자본 수익률**

- 투입한 돈 대비 수익률 (%)

**계산 공식:**

```
ROE = (현재 손익 / 투입 자금) × 100
```

**예시 (ETH):**

- 손익: +$7.61
- ROE: +5.6%
- 의미: 투입 자금의 5.6% 수익

**레버리지 효과:**

```
가격 변동: +0.56%
레버리지: 10배
ROE: +5.6% (10배 확대)
```
